# System Monitor CLI Tool 💻

## Overview

A command-line system monitoring tool written in Python.

It displays real-time system statistics such as CPU usage, RAM usage, disk usage, and network activity.

## Features

* CPU usage monitoring
* RAM usage monitoring
* Disk usage monitoring
* System uptime display
* CPU temperature monitoring
* Network statistics
* Top processes by CPU usage

## Improvements

* Resource usage trend indicators (↑ ↓ →)
* Reduced terminal flickering by updating values dynamically

## Technologies

* Python
* psutil
* Linux system metrics
