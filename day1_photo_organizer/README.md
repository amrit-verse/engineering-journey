# Photo Organizer 📷

## Overview

This tool organizes photos and videos automatically based on their EXIF metadata.

It helps clean and structure large photo collections.

## Features

* Reads EXIF metadata from images
* Sorts photos by date
* Automatically organizes files into folders

## Technologies

* Python
* EXIF metadata
* Command line tools

## Example Use Case

Organizing thousands of photos exported from a smartphone.
